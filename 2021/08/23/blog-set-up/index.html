个人博客搭建：WordPress与Hexo（Github与Gitee），如何选择？

发表于 2021-08-23 分类于 博客搭建

本文比较两种主流博客–WordPress与Hexo，其中Hexo博客搭建又提供了Github和Gitee两个部署平台，其中总能pick到你需要的那一款~

这篇文章只是用表格教你在这几种之间如何选择，搭建教程上网查找，以后可能会写。

每个比较项目中表现最优秀的将被加粗

搭建方式 WordPress+境外VPS Hexo+Github Hexo+gitee
成本 相对较高（折合大约￥30月起步，域名+服务器） 如不需自定义域名则免费（自定义域名一年￥50-￥150不等，许多商家都有首年甚至好几年的优惠） 全免费
是否支持自己的域名 必需 支持或xxx.github.io 不支持,xxx.gitee.io
访问速度 境内vps：快；境外vps：一般 慢 快
写作方式 在线 本地 本地
部署 即时动态 远程，但可以自动生成网页 远程，而且不可以自动生成网页
类型 动态 静态 静态
搭建难易度 简单 有难度 有难度
写作难易度 简单 需要掌握markdown 需要掌握markdown
博文撰写是否支持手机 支持 不支持 不支持
自定义程度 高 有限 有限
自定义难易度 简单 复杂 复杂
应用场景 多（世界上30%的网站都是用WP搭建的，比如……白宫官网） 个人博客为主 个人博客为主
主题数量 五花八门 不少，但是NexT最流行 不少，但是NexT最流行
知识储备 几乎无 Markdown语法+熟练的电脑操作（最好有点编程基础） Markdown语法+熟练的电脑操作（最好有点编程基础）
评论功能 自带 需要安装第三方 需要使用第三方
功能 插件众多，自然五花八门 不多，但作为个人博客完全足够 不多，但作为个人博客完全足够
性能 理论上低，如果肯氪金性能也会很好 高 高
是否占用本地空间 否 是 是
是否备案 境外vps无需备案；境内必需备案 无需 无需

最后再说几点：

1.愿意为个人博客花点钱的直接WordPress。

2.在Github和Gitee的选择上，如果你需要自定义域名只能Github，不然直接Gitee

。虽然Gitee有空间限制，但是足够用。。国内访问Github太慢了，不稳定还容易被墙。。但是Gitee Pages每次部署完是不支持自动生成网页的，需要到它官网上手动点一下更新，这也比较蛋疼。本人不喜欢Coding，但也许它是一个更好的选择。

3.如果对电脑操作还不熟悉的就先别建Hexo博客了

4.Hexo博客搭建过程中经常会遇到一些莫名奇妙的报错。。一般网上都能找到答案。。搭建过程会有一点点繁琐。

5.WP博客搭建上，境内vps必需备案，备案流程极其繁琐，而且还有失败率，网站还要受到严格信息监管。但境内速度极快。如果不特别在意速度，推荐境外vps，商家推荐vltur和hostwinds，大部分商家提供的节点都是美国西岸快些。注意有可能被墙，而hostwinds可以免费更换ip以应对。

6.Markdown编译器个人喜欢Typora,虽然没有正式版（正因如此，所以免费）
